# Mobile Mental Health Express

This repository contains a simple static landing page for **Mobile Mental Health Express** — an informational front page intended for a mobile mental health outreach program.

Files
- `index.html` — the main static HTML page for the project.

How to view locally

1. From the repository root open the file in a browser, or run a quick local server:

```bash
cd /workspaces/business-proposal
python3 -m http.server 8000
# then open http://localhost:8000/index.html
```

Publishing

This project is hosted on GitHub at: https://github.com/jayveedimapilis07-cmyk/business-proposal
